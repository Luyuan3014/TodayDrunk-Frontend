# TodayDrunk（今日小酌）

现代化饮酒记录应用 - 前端项目

## ✨ 功能特色

- 🍻 **饮酒记录** - 详细记录每次饮酒体验
- 📊 **数据分析** - 可视化统计和趋势分析
- 🗺️ **地图功能** - 饮酒地点标记和分享
- 📚 **知识库** - 酒类知识和文章
- 🏆 **成就系统** - 解锁各种饮酒成就
- 👤 **个人中心** - 用户信息和偏好设置

## 🛠️ 技术栈

- **前端框架**: React 18 + TypeScript
- **样式方案**: TailwindCSS
- **状态管理**: Zustand
- **路由**: React Router
- **图表**: Chart.js + react-chartjs-2
- **图标**: Lucide React
- **构建工具**: Vite

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/Luyuan3014/TodayDrunk-Frontend.git

# 安装依赖
cd TodayDrunk-Frontend
npm install

# 启动开发服务器
npm run dev
```
